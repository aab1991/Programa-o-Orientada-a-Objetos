# Programação-Orientada-a-Objetos
# Atividades em Java -- README

Este repositório contém uma série de exercícios desenvolvidos em Java,
cada um abordando conceitos essenciais da linguagem, como entrada de
dados, cálculos, estruturas condicionais e loops. Abaixo você encontra a
descrição de cada atividade e instruções gerais para execução.

------------------------------------------------------------------------

##  1. Calculadora de Desconto

Programa que recebe: - Valor de um produto - Porcentagem de desconto\
Calcula e exibe: - Valor do desconto - Preço final após aplicação do
desconto\
**Entrada:** Scanner\
**Execução:** `java CalculadoraDesconto`

------------------------------------------------------------------------

##  2. Calculadora de ITBI

Recebe via JOptionPane: - Valor de transação de um imóvel - Valor
venal - Porcentagem do ITBI\
Calcula: - Imposto sobre o maior valor entre transação e venal\
**Execução:** `java CalculadoraItbi`

------------------------------------------------------------------------

##  3. Sistema de Avaliação

Usando JOptionPane, recebe: - Nota da Prova 1 - Nota da Prova 2 - Nota
do Trabalho\
Calcula a média e informa: - Aprovado (média ≥ 6) - Reprovado\
**Execução:** `java SistemaDeAvaliacao`

------------------------------------------------------------------------

##  4. Calculadora de Aposentadoria

Recebe: - Idade - Sexo - Anos de contribuição\
Determina: - Se a pessoa já pode se aposentar - Quantos anos faltam\
Baseado nas regras: - Aposentadoria por idade\
- Aposentadoria por tempo de contribuição\
**Execução:** `java CalculadoraDeAposentadoria`

------------------------------------------------------------------------

##  5. Circuito de Resistências

Recebe quatro resistências ligadas em série e calcula: - Resistência
equivalente - Maior resistência - Menor resistência\
**Execução:** `java CircuitoResistencias`

------------------------------------------------------------------------

##  6. Sistema de Login

Simula login com: - Usuário fixo: `java8` - Senha fixa: `java8` O
usuário tem **3 tentativas**, com mensagens de erro a cada falha.\
**Execução:** `java SistemaLogin`

------------------------------------------------------------------------

##  7. Gerador de Tabuada

Recebe um número e apresenta sua tabuada de 1 a 10 usando loop.\
**Execução:** `java GeradorTabuada`

------------------------------------------------------------------------


##  Requisitos

-   JDK 8 ou superior
-   IDE opcional (NetBeans, Eclipse, IntelliJ) ou uso do terminal
